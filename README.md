# Deep Live Cam - Traducción al Español
Este repositorio contiene la versión traducida al español del proyecto Deep-Live-Cam (interfaz y mensajes) sin renombrar identificadores del código.
Instrucciones rápidas:
1. Clonar el repositorio.
2. Instalar dependencias requeridas (opencv, insightface, opennsfw2, customtkinter, etc.).
3. Ejecutar: `python run.py --lang es` para iniciar la UI en español.
